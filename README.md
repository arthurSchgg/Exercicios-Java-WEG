## 🛠️ Tecnologias

- **Java** (JDK 11+)
- **Eclipse IDE** / **IntelliJ IDEA**
- Paradigma de **Programação Orientada a Objetos**

---

## 📖 Conceitos Abordados

- Lógica de programação e estruturas de controle
- Arrays, vetores e matrizes
- Programação Orientada a Objetos (classes, herança, polimorfismo, encapsulamento)
- Tratamento de exceções (`try-catch`)
- Estrutura em camadas (model / service / view)
- Estatística e probabilidade aplicadas

---

## ▶️ Como Executar

1. Clone o repositório:
```bash
   git clone https://github.com/seu-usuario/exercicios-java-weg.git
```
2. Abra a pasta do projeto desejado no Eclipse ou IntelliJ.
3. Execute a classe `Main` correspondente.
